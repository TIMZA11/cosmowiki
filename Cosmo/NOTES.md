### Решение по Wikipedia API
Используем комбинацию двух эндпоинтов:
1. `rest_v1/page/summary/` — для быстрого получения описания и thumbnail
2. `w/api.php?action=parse` — для получения инфобокса с параметрами объекта

CORS поддерживается через параметр `origin=*` в MediaWiki API.

### Интеграция NASA Images API
- Эндпоинт: `https://images-api.nasa.gov/search?q={query}&media_type=image`
- Бесплатный, без API-ключа, поддержка CORS
- Создан маппинг 70+ объектов каталога на англоязычные поисковые термины NASA
- Используется как fallback для изображений: сначала Wikipedia, если нет — NASA
- На карточках с NASA-фото отображается бейдж «NASA»
- Кэширование в localStorage на 24 часа

### Интеграция Naked Science
- Источник: https://naked-science.ru/article/top/gid-po-vselennoi-kto-est
- Извлечены описания для 8 категорий + 6 типов объектов
- Найдены 13 реальных URL иллюстраций (через data-lazy-src в HTML)
- Показывается как врезка-блок на странице категории

### Этап полировки (безопасность, данные, анимации)
**XSS-исправления:**
- object.js: полная переработка — все innerHTML заменены на DOM API (createElement/textContent)
- object.js: inline onerror/onclick → addEventListener + event delegation
- category.js: карточки объектов рендерятся через DOM API вместо innerHTML с интерполяцией
- Добавлен escapeHtml() для всех значений из API
- Добавлен encodeURIComponent() для URL-параметров

**Исправления данных:**
- Расстояния спутников Юпитера: Ио 588.3 млн км, Европа 591.1 млн км, Ганимед 594.6 млн км (были все 628.3)
- Расстояния спутников Сатурна: Титан 1222 млн км, Энцелад 1238 млн км (были оба 1270)
- NASA-запросы для квазаров: apm08279 → 'APM 08279 quasar gravitational lens', ton618 → 'TON 618 quasar black hole'
- NASA-запросы для пульсаров: улучшены с 'pulsar' на конкретные термины

**travel-time.js:**
- formatLightDistance: заменён хардкод 365.25*24*60 на HOURS_PER_YEAR*60

**Анимации:**
- Intersection Observer для секций страницы объекта (hero, travel, params, gallery)
- CSS section-animate с плавным fadeInUp и каскадной задержкой
- Naked Science блок на странице категории с анимацией появления

**Обработка ошибок:**
- category.js: try/catch вокруг loadObjects, .catch() на каждом API-запросе
- object.js: try/catch + индивидуальные .catch() (было ранее)
